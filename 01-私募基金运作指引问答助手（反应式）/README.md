


# 私募基金运作指引问答助手（反应式）

---

## 一、项目背景与目标

随着私募基金行业的快速发展，法规、合规和业务规则日益复杂。很多从业者、投资者、合规人员甚至新员工，常常会遇到“某条规则怎么规定？”、“合格投资者标准是什么？”、“有哪些费用？”等问题。

本项目旨在用AI技术，打造一个“能随时回答私募基金规则问题的智能机器人”，让任何人都能像查字典一样，快速、准确、无门槛地获得权威答案。

---

## 二、项目简介

本项目（FUND Q&A ASSISTANT - REACTIVE）是一个基于LangChain与大语言模型（LLM）的私募基金智能问答系统。它内置22条最常用的私募基金规则，支持自然语言提问，自动理解你的问题并给出专业答案。

**适用人群：**
- 金融从业者、基金公司员工
- 法务、合规、风控人员
- 投资者、理财顾问
- 金融AI/NLP学习者
- 新员工、实习生
- 任何想快速了解私募基金规则的人

## 四、核心功能与特性

## 五、系统原理与技术细节

### 1. 规则知识库（知识底座）
- 内置22条私募基金核心规则，每条规则包含唯一ID、分类、标准问法、详细答案、关键词列表。
- 支持规则扩展与自定义。

### 2. 多级智能匹配（如何理解你的问题）
- **特殊处理器**：针对高频、歧义性强的问题，优先精准路由。
- **关键词权重系统**：40+关键词，分4-10分等级，自动加权匹配。
- **语义理解与LLM辅助**：对复杂、口语化、变体问题，调用LLM进行语义补全与推理。

### 3. 交互与测试（怎么用、怎么验证）
- **命令行交互**：直接运行主程序，输入自然语言问题实时获得答案。
- **快速测试**：test_qa_assistant.py，7个关键用例，验证系统主流程。
- **综合测试**：test_comprehensive.py，67个真实用户问题，覆盖所有规则和多种问法。

### 4. 技术栈（用到哪些技术）
- Python 3.8+
- LangChain 1.2.0+
- langchain-community
- dashscope（通义千问API）
- pydantic

- 标准问法：“私募基金的合格投资者标准是什么？”
- 关键词：“合格投资者标准”
- 口语化：“想投私募基金需要什么条件？”
- 场景化：“公司投资私募基金有什么要求？”

**Q2：系统能保证100%准确吗？**
- 快速测试（7个典型问题）100%通过。
- 综合测试（67个真实问题）覆盖全部规则，准确率高。
- 对于极度口语化或新颖表达，建议补充关键词或规则。

**Q3：怎么扩展新规则或优化效果？**

- 在 keyword_weights 字典里加新关键词或调整分数。
- 在 process_query 方法里扩展特殊处理逻辑。
- 在 test_comprehensive.py 里加新测试用例。
